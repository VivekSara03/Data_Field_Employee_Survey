# üìä Employee Morale & Inclusivity Analysis ‚Äì Data Field Survey

## üéØ Problem Statement

This project focuses on a **prescriptive analysis** of employee morale and inclusivity within the data industry. The objective was to understand how professionals in the data field perceive work-life balance, inclusivity, learning opportunities, and upward mobility.

The analysis was designed to support **HR and Management Team** in evaluating workforce sentiment and identifying diversity trends within the industry.

### Key KPIs Analyzed:

* Work-Life Balance Score
* Upward Mobility Perception
* Co-worker Inclusivity Score
* Satisfaction with Learning Opportunities

---

## üìÇ Dataset Information

* Source: Dataset shared via Alex The Analyst YouTube channel
* Records: ~630 employees
* Data Type: Mixed (Categorical + Numerical)

### Variables Included:

* Age
* Gender
* Ethnicity
* Salary
* Experience Level
* Job Role
* Satisfaction & Inclusivity Ratings

### Data Preparation:

The dataset was unstructured and required significant cleaning:

* Handled null values (replaced with logical assumptions where possible, otherwise labeled as ‚ÄúNot Mentioned‚Äù)
* Corrected inconsistent entries
* Performed data type standardization
* Structured the dataset for analysis in Power BI

## üìà Key Insights

* The gender ratio in the data field is approximately **1:3 (Female:Male)**.
  In Data Engineering roles, the ratio drops significantly to nearly **1:8.5**.

* The average age of professionals in the data field is **30 years**, indicating a relatively young workforce.

* Representation of **White/Caucasian professionals** is significantly higher compared to other ethnic groups.

* The overall average inclusivity score across genders was **5.76/10**, suggesting moderate but improvable inclusivity levels.

---

## üîç Business Implication

While this project was exploratory in nature, the findings highlight:

* Gender imbalance in technical roles
* Limited diversity across certain demographics
* Opportunities for improving inclusivity initiatives
